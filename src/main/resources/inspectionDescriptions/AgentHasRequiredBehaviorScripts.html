Detects missing required event scripts for BHVR value

Certain BHVR bitflags require certain scripts to be present for agent inject.

Example: 


new: simp 2 21 45553 "blnk" 1 0 0
bhvr 11

bhvr 11 = (activate 1, activate 2, hit)
Inspection will show warning if scripts 1 (activate 1), 2 (activate 2) or 3 (hit) are missing
